{{< /blocks/cover >}} {{% blocks/lead %}} It all started with [The Phoenix Flavour](/tpf/introduction/), a manual modding guide that later became a Wabbajack setup. Since TPF, I have branched out and created the Wabbajack lists [Legends of the Frost](/lotf/introduction/) for Skyrim SE as well as [Welcome to Paradise](/wtp/introduction/) for Fallout 4. In addition to my own projects, this website features other Wabbajack lists that were based on TPF and LOTF. Find out more below. {{% /blocks/lead %}} {{< blocks/section >}} {{% blocks/feature icon="fa-github" title="Legends of the Frost" url="/lotf/introduction" %}} Legends of the Frost is a Wabbajack list for Skyrim SE, strictly limited to fixing issues in the base game without changing it significantly for more enjoyable gameplay that is close to the vanilla experience. {{% /blocks/feature %}} {{% blocks/feature icon="fa-github" title="The Phoenix Flavour" url="/tpf/introduction/" %}} The Phoenix Flavour is a manual guide and Wabbajack list for Skryim SE, focusing on improved visuals and gameplay without compromising the game's original vision. {{% /blocks/feature %}} {{% blocks/feature icon="fa-github" title="Welcome to Paradise" url="/wtp/introduction/" %}} Welcome to Paradise is a
